## 🚀 Cara Menjalankan (3 Langkah)

### Langkah 1: Database Setup (Pilih Salah Satu)

**MySQL (Recommended):**
```bash
mysql -u root -p
CREATE DATABASE spk_bansos;
EXIT;
```

**Atau SQLite (Already configured):**
```bash
# Tidak perlu - sudah siap!
```

### Langkah 2: Terminal 1 - Backend
```bash
php artisan migrate --force
php artisan db:seed
php artisan serve
```

Tunggu sampai muncul: `Server running at http://127.0.0.1:8000`

### Langkah 3: Terminal 2 - Frontend (Tab/Window baru)
```bash
npm run dev
```

Tunggu sampai muncul: `➜ Local: http://localhost:5173/`
